# Web Based Student Management System

Minor Project for the Bachelor of Computer Applications (BCA) program.

## Description
The Web Based Student Management System is a web application developed to manage student records digitally. It allows administrators to add, view, update, and delete student information through a centralized platform.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL

## Features
- Admin login
- Student registration and record management
- Simple and user friendly interface

## Project Structure
- app.py: Main application file
- templates/: HTML templates
- static/: CSS and JavaScript files
- database/: Database schema

## Note
This project is developed strictly for academic purposes.
